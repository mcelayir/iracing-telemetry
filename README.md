# iRacing Rust Telemetry

A lightweight, high-performance telemetry view for iRacing.

## Features
- [ ] Real-time RPM, Speed, and Gear display
- [ ] Low CPU/RAM footprint (Rust-based)
- [ ] WebSocket bridge for custom overlays (Coming Soon)
- [ ] Cloud data upload (Coming Soon)

## Technical Stack
- **Language:** Rust
- **Library:** Pitwall (iRacing SDK)
- **Runtime:** Tokio (Async)

## How to Build
1. Install Rust (Windows)
2. Run `cargo build --release`
